Welcome To The

A community oriented Minecraft server dedicated to the Resonym stream and Discord community.

Rules:

The Minecraft server is an extension of the Resonym community, this means that you held to the same standards in the Minecraft server that you are held to in other Resonym community spaces. For exact rules you can see the #rules channel in Resonym Discord

Getting Access:

Before you can properly connect to the server, you will need to submit your Minecraft username so we can add you to the list of allowed players. Without this you will not be able to connect to the server at all.

Connecting:

There are steps on how to connect to the server using both Bedrock and Java editions of the game.
